
# Tiktok Video Downloader API

Download video from Tiktok by this API

## API Reference


```http
  GET /tiktok?url={tiktok_video_url}
```

| Parameter | Type     | Description                |
| :-------- | :------- | :------------------------- |
| `url` | `string` | **Required**. Your Video Link |
